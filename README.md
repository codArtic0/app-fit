# App-fit

Um aplicativo em desenvolvimento em Python para monitoramento pessoal de hábitos fitness. 

A princípio, faz cálculo de Taxa Metabólica Basal (TMB), IMC e distribuição personalizada de macronutrientes.

Este código está aberto a colaborações.

Você pode conferir o protótipo aqui: https://www.figma.com/proto/rTnURBdNNkwqrjZYyij55N/FIT-FORGE---MONITORAMENTO-FITNESS?node-id=1-2&t=rxhRst33LaxoHCtr-1 (em desenvolvimento)
